== Blog configuration is a code

.The content of `_config.yml` file
[source,yaml]
----
# Date / Time format
date_format: YYYY-MM-DD
time_format: HH:mm:ss

# Pagination
per_page: 10
pagination_dir: page

# Extensions
theme: mytheme

disqus_shortname: eprintstacktrace

# Deployment
deploy:
  type: git
  repo: git@github.com:wololock/wololock.github.io.git
  branch: master

all_minifier: true
----
